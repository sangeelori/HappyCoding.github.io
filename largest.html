Program to find the 3rd largest numbers

Description:

Create an integer array of capacity 50. Take size of the array as input from users. Using for loop, take array element as input from users and insert into the array. After inserting all the elements of the array, consider the very first element of array to be the largest. Run a for loop, from 1 to arraySize-1, extracting array element one by one and comparing it to the largest element. If largest element is smaller than the comparing element of array, then the largest element is updated to current element of array. At the end, the largest element will hold the actual largest value present in the array.

Code in Java

public class ThirdLargestInArrayExample{
public static int getThirdLargest(int[] a, int total){
int temp;
for (int i = 0; i < total; i++)
{
for (int j = i + 1; j < total; j++)
{
if (a[i] > a[j])
{
temp = a[i];
a[i] = a[j];
a[j] = temp;
}
}
}
return a[total-3];
}
public static void main(String args[]){
int a[]={1,2,5,6,3,2};
int b[]={44,66,99,77,33,22,55};
System.out.println("Third Largest: "+getThirdLargest(a,6));
System.out.println("Third Largest: "+getThirdLargest(b,7));
}}


Output:

Third Largest:3
Third Largest:66
